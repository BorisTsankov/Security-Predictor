# Airport Security Time Prediction

This project predicts total passenger processing time at an airport security checkpoint using machine learning regression models.

## Structure

- `data/raw/` raw input data
- `data/processed/` cleaned datasets
- `notebooks/` Jupyter notebooks for analysis and modeling
- `models/` saved trained models
- `reports/figures/` exported figures for the report

## Main notebooks

1. `01_data_understanding.ipynb`
2. `02_preprocessing_and_features.ipynb`
3. `03_modeling.ipynb`
4. `04_evaluation.ipynb`
